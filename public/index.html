Help & Documentation

üìñ Overview

Nmap GUI is a professional web-based network scanning tool that provides an intuitive interface for Nmap operations with advanced analysis capabilities.

üéØ Scan Modes Explained

  • Ping Scan (-sn): Discovers active hosts without port scanning
  • Quick Scan (--top-ports 200): Scans the 200 most common ports
  • Full TCP Scan (-p-): Scans all 65,535 TCP ports (time-intensive)
  • Service Detection (-sV): Identifies services and versions running on open ports
  • OS Detection (-O): Attempts to determine the target operating system
  • NSE Scripts (-sC): Runs default NSE scripts for deeper analysis

üí° Best Practices

  • Start with Ping scan for host discovery
  • Use Quick scan for initial reconnaissance
  • Run Full scan only when necessary (very time-consuming)
  • Always have authorization before scanning targets
  • Export results for documentation and analysis
  • Use filters to focus on specific ports or services

‚öôÔ∏è Advanced Tips

  • Root/Admin privileges required for SYN scans (-sS)
  • OS detection works better with root privileges
  • CSV exports can be analyzed in spreadsheet applications
  • XML files preserve all scan details for future reference
  • Scan history persists in browser storage
